# WellesleyFreshPlus

Wellesley Fresh + is an upgraded version of the current Wellesley Fresh with 3 added functionality:
1. Felp: this help the user get a better sense of the comment and the real pictures of the dining hall food
2. Dish Dash: this functionality is a forum where students could discuss
3. Wellesley Feast: this is a notification bar where people can post notices of food event on campus
